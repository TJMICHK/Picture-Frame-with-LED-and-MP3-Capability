# Picture Frame with LED and MP3 Capability
The goal of this project is to make a hanging collage picture frame with back-drop LED and sound. This project is meant to be a gift, due before the date of March 14th. This project will use an one ESP32, < 1m of LED Lighting, one MP3 Module, 2 Speakers (3W 8Ohm), and 1 PAM Speaker Amplifier Module.
